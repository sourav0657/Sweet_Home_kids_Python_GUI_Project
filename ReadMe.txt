ReadMe
-------
Project Overview
----------------

This is a Python-based project developed using the Tkinter library for GUI and the webbrowser module to play educational videos. The project is designed for child learning purposes, providing interactive visual elements that play learning videos when clicked.

How to Run the Project:
----------------------

1. Download the project files to your local workstation.
2. Make sure Python is installed on your system.
3. Open the project using PyCharm or Visual Studio Code.
4. Run the file named sweet_home.py.

Project Components:
-------------------

The scene includes the following elements:

1. Sun/Moon
2. Bushes
3. House
4. Pond

Interactive Features:-
--------------------
1. Home Door:-  Click on the Home Door to open a video that helps kids learn the ABC.
2. Sun:- Click on the Sun to switch the scene to night mode. At the same time, a video explaining the Sun will start playing.
3. Moon:- Click on the Moon to switch the scene back to day mode. A video explaining the Moon will also play.
4. Pond:- Click on the Pond to watch a video that explains about a pond.
